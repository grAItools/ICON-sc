# archive/ — superseded documents

Documents with no ongoing relevance, kept for reference; nothing here is
authoritative. The archive accepts dead documents of **any kind** — proposals, specs,
plans, reports, policies, ADRs — not only work documents; arrivals keep their dying
names, kebab-cased (scheme-exempt, ADR-0006; case rule TD-51.1, ADR-0007). Each file carries a supersede note above its
original text naming what replaced it. History and original context:
`git log --follow <file>`.
