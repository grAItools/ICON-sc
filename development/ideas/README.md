# ideas/ — future proposals and phase outlines

Proposals for future development, including the migrated post-slice phase outlines
P2–P7 (`PN_<snake>.md`; new ideas take a descriptive snake_case name). An idea is
living until graduated: it is specced into `development/specs/` steps and then stays
as a frozen record of the original scope.

**`Status:` header convention** — every idea carries, above its title:
`proposed / accepted / accepted-roadmap / rejected / superseded / graduated → spec SXX`.
